# ZSGate (OSS)

Enterprise LLM Gateway (community edition).

## What you get in OSS
- Multi-vendor unified gateway APIs
- Model alias + weighted routing + failover
- Basic audit and usage analytics
- Team-level statistics by user/project/department

## Quick run
```bash
make run-control
# new terminal
make run-data
```

## Build
```bash
make build
```

## Demo flow
```bash
bash scripts/demo.sh
```
